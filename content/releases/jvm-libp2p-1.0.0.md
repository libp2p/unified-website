This release contains experimental [yamux](https://github.com/libp2p/specs/tree/master/yamux) support as well as various other improvements and dependency updates. From this version onwards, the dependency published will be `io.libp2p:jvm-libp2p`. For previous versions of the library, the old `io.libp2p:jvm-libp2p-minimal` dependency can be used.
